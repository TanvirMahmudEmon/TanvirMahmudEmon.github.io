## How to Create a Python Virtual Environment in Windows
### Install Virtualenv:
```pip install virtualenv```

- Checkout command:
  - ```pip freeze```
  - Example output: virtualenv==16.4.1
  
 ### Make a directory:
 ```mkdir mydir```
 
 ```cd mydir```
 
 ### Now Create a virtualenv:
 ```virtualenv . ```
 
 ### Activate the virtualenv:
 ```.\Scripts\activate```
 
 ### To Deactivate the Virtualenv:
 ```deactivate```
